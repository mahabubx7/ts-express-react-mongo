```json
{
  "backend": "express",

  "runtime": "node.js (with TypeScript)"
}
```

### Run this application

Make sure you have a `.env` file in the root folder. And,

```env
API_PORT=4000
API_HOST=0.0.0.0
MONGODB_URI=mongodb://localhost:27017/ermovied
```

**Now, run this command** [`yarn` is recommended]

```bash
$ yarn start
```
